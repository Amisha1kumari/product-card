# ☕ Coffee Mug Product Card

A responsive product card UI built using HTML and CSS. The project showcases a modern e-commerce card design featuring a product image, description, pricing section, and an interactive call-to-action button.

## 📸 Preview

This project displays a stylish coffee mug product card with:

* Product image
* Product category
* Product title
* Product description
* Discounted price and original price
* Add to Cart button

## 🚀 Features

* Clean and modern user interface
* Responsive card layout
* Flexbox-based design
* Rounded corners and elegant spacing
* Product pricing section
* Interactive button styling
* Beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
